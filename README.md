# 🔍 Port Scanner GUI

A simple Python GUI-based Port Scanner built using Python socket programming and Tkinter.

## ✨ Features
- Scan ports for a target IP
- GUI-based interface
- Input validation
- Displays open ports

## 🚀 How to Run

```bash
python port_scanner.py
```

## 🛠 Tech Stack
- Python
- Tkinter
- Socket Programming

## ⚠️ Disclaimer
Use only on systems you own or are authorized to test.
